# Гра "Шашки"

## Опис функціональності

### Інтерфейс користувача (UI)
1. Головне меню:
    - Вибір противника (ШІ або людина)
    - Вибір кольору шашок (чорні або білі)
    - Вихід з програми
2. Меню під час гри:
    - Нова гра
    - Збереження гри
    - Завантаження останнього збереження
    - Вихід з програми
3. Ігровий екран:
    - Відображення гральної дошки 8х8
    - Відображення шашок двух гравців
    - Підсвічування можливих ходів вибраної шашки
    - Повідомлення про чергу ходу
    - Повідомлення про те, хто переміг
4. Екран завершення гри:
    - Відображення результату гри
    - Кнопка для початку гри заново або повернення в меню

### Ігрова логіка
1. Початок гри:
    - Ініціалізація дошки та шашок
    - Випадкове або ручне визначення першого гравця
2. Механіка гри:
    - Рух шашок по діагоналі
    - Можливість "бити" шашки суперника
    - Автоматичне перетворення в дамку при досягненні останнього ряду
    - Діє правило "Бій обов'язковий"
    - Ходи дамкою в обидві сторони
    - Зміна черги ходу між гравцями
3. Перевірка завершеності гри:
    - Перемога при відсутності шашок у суперника або неможливості ходу
    - Нічия при патовій ситуації
4. Гра з комп'ютером:
    - Простий AI: вибір допустимого ходу за пріоритетом взяття

### Збереження даних
- Збереження у файл (позиція кожної шашки, тип і черга ходу)
- Завантаження гри при запуску або з головного меню
- Автозбереження при виході з гри

### Додаткові можливості
- Режим «грати з другом» на одному ПК
- Скасування ходу (можливість повернутися на кілька кроків назад)
- Гра проти комп’ютера (реалізація простого AI з базовими стратегіями)
